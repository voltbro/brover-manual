Для первичной проверки и работы с роботом вы можете открыть встроенный веб-интерфейс.

В браузере перейдите по одному из адресов:

* `http://broverNN.local:8080`, где `NN` — номер вашего робота;
* `http://<IP-адрес_робота>:8080`, если вы используете IP-адрес.

На открывшейся странице отображаются основные данные о местоположении робота, напряжение питания, а также видеопоток с установленной камеры (см. скриншот ниже).

![](/img/intro/web.png) 

Управление роботом осуществляется с клавиатуры с помощью клавиш **W**, **S**, **A**, **D**.

Также из веб-интерфейса можно перейти в **Web VS Code**, работающий непосредственно на роботе. Для этого нажмите соответствующую вкладку в левом верхнем углу страницы.

**Пароль для входа:**

```
brobro
```

Кроме того, в Web VS Code можно попасть напрямую, открыв в браузере адрес:

* `http://broverNN.local:8090`
* или `http://<IP-адрес_робота>:8090`

Чтобы сменить пароль от Web VS Code на роботе отредактируйте файл  `/home/pi/.config/code-server/config.yaml`
