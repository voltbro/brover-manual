Ровер представляет собой шестиколёсную мобильную платформу с модульной конструкцией. Корпус выполнен из оргстекла, несущий каркас — из алюминиевых профилей, обеспечивающих жёсткость конструкции при сравнительно малой массе.

Платформа оснащена шестью колёсами, установленными на подвеске типа *rocker-bogie*. Такая схема подвески позволяет роверу эффективно преодолевать неровности рельефа, бордюры и препятствия, сохраняя устойчивость и контакт всех колёс с поверхностью.

Каждое колесо приводится в движение отдельным *DC-мотором*. **Управление моторами** осуществляется с помощью шести плат [**VBCore DC Motor Driver**](https://docs.vbcores.ru/docs/Hardware/dc-driver), каждая из которых построена на драйвере Allegro A4955 и работает под управлением модуля [**VBCore**](https://docs.vbcores.ru/docs/Hardware/vbcore). Все драйверы объединены в общую шину *CAN*, обеспечивающую надёжный и масштабируемый обмен данными.

**Связь между драйверами моторов и бортовым компьютером** осуществляется через плату [**CAN FD Raspberry Pi HAT**](https://docs.vbcores.ru/docs/Hardware/canfd-hat), подключённую к *Raspberry Pi 5*. Над *CAN-адаптером* установлена [**коммутационная плата питания**](https://docs.vbcores.ru/docs/Hardware/ch-quick-connect), предназначенная для удобного *подключения пользовательских внешних устройств* к роверу. Подключаемые устройства могут быть интегрированы в общую *CAN-шину* и обмениваться данными с бортовым компьютером Raspberry Pi.

Под *Raspberry Pi* установлен 9-осевой [**IMU-модуль**](https://docs.vbcores.ru/docs/Hardware/bhi360) с абсолютной ориентацией, построенный на базе датчика Bosch BHI360, используемый для оценки ориентации ровера. Рядом расположен адаптер для приёма сигналов радиомодельных джойстиков по протоколу *SBUS*. В комплекте с ровером поставляется радиопульт **Radiolink**, позволяющий осуществлять ручное управление платформой.

Ровер также **оснащён камерой**, предназначенной для получения визуальной информации об окружающей среде и обеспечения дистанционного управления ровером через веб-интерфейс на основе видеопотока. Для контроля и управления бортовой системой электропитания установлена специализированная [**плата PowerBoard**](https://docs.vbcores.ru/docs/Hardware/powerboard).

На корпусе ровера размещены:
- кнопка включения питания;
- пользовательская программируемая кнопка;
- большая аварийная кнопка экстренной остановки.

При нажатии аварийной кнопки питание драйверов двигателей немедленно отключается, что приводит к остановке ровера, при этом бортовой компьютер *Raspberry Pi* продолжает работать.
