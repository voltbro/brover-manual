# Веб-интерфейс

Веб-интерфейс позволяет управлять ровером из браузера и просматривать основные данные без установки дополнительного ПО.

Через веб-интерфейс можно:

* управлять ровером с клавиатуры;
* получать видеопоток с камеры;
* просматривать данные одометрии.

Видеопоток в веб-интерфейсе формируется из ROS-топиков камеры. За получение изображения отвечает нода `/camera1`, а за публикацию потока по HTTP — `/web_video_server`.

## Требования

Компьютер и ровер должны находиться в одной Wi-Fi-сети.

## Подключение по имени хоста

Откройте в браузере:

```bash
http://broverXX.local:8080
```

где `XX` — номер ровера, указанный на корпусе Raspberry Pi.

Пример:

```bash
http://brover15.local:8080
```

## Подключение по IP-адресу

Если имя хоста не открывается, используйте IP-адрес ровера:

```bash
http://<IP-адрес ровера>:8080
```

Пример:

```bash
http://192.168.1.115:8080
```

Если IP-адрес неизвестен, найдите его в интерфейсе роутера или через сетевой сканер. Подробные способы описаны в разделе [SSH-подключение](20-ssh-connection.md#определение-ip-адреса-ровера).
