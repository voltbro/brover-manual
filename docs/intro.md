# BRover-E5

**BRover-E5** — это модульный шестиколёсный образовательный робот-ровер, разработанный для изучения современной робототехники на практике.

Платформа сочетает в себе возможности учебного набора и полноценной инженерной системы. Она подходит как для школьников и студентов, которые только начинают знакомство с робототехникой, так и для более продвинутых пользователей, которым важно понимать внутреннее устройство и архитектуру роботизированных систем.

Ровер разработан на базе электроники [**VBCores**](https://vbcores.ru/), построен на **Raspberry Pi 5**, использует **CAN-шину** для связи между модулями и полностью поддерживает **ROS 2**. Это делает его удобным инструментом для изучения Linux, принципов работы ROS 2, а также взаимодействия распределённых устройств в составе одной системы.

## Назначение

**BRover-E5** предназначен для обучения и практической работы. Он может использоваться в школах, вузах и инженерных кружках для проведения лабораторных работ, выполнения учебных проектов и освоения современных подходов к разработке робототехнических систем.

Ровер поддерживает несколько режимов работы: ручное управление (например, с джойстика), программное управление через Python и ROS 2, а также полуавтономные сценарии, в которых пользователь может реализовать собственную логику поведения робота.

## Особенности платформы

Архитектура **BRover-E5** построена по модульному принципу. Высокоуровневая логика — обработка пользовательских команд, работа с алгоритмами и автономными режимами — выполняется на Raspberry Pi. При этом управление приводами и периферией вынесено на отдельные микроконтроллерные модули, связанные между собой по CAN-шине.

Такой подход позволяет не только повысить надёжность системы, но и наглядно демонстрирует принципы построения современных робототехнических платформ.

Ровер изначально спроектирован как расширяемая платформа. Пользователь может подключать дополнительные устройства и модули, интегрировать их в систему и обеспечивать обмен данными через CAN и ROS 2.
