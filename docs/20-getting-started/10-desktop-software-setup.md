# Настройка ПО рабочего места

Для комфортной работы с **BRover-E5** необходимо подготовить компьютер. В зависимости от уровня подготовки и задач доступны три варианта: быстрый старт на Windows, работа через виртуальную машину и полноценная работа в Linux.

## Вариант 1. Windows (быстрый старт)

Этот вариант подходит, если нужно быстро подключиться к роверу и начать работу без сложной настройки.

Основной инструмент — терминал для подключения по SSH.

В Windows можно использовать:

* встроенный **Windows PowerShell**
* сторонние SSH-клиенты (например, Termius или PuTTY)

![alt text](img/start-command-promt.png)

Дополнительно можно установить **Visual Studio Code** и настроить подключение к роверу через **Remote-SSH**. Это удобно, если не хочется работать через встроенный web-интерфейс code-server.

Такой вариант позволяет выполнять базовые операции: подключение к роверу, запуск команд и работа с файлами.

Ссылка на официальный сайт: [`Скачать VS Code`](https://code.visualstudio.com/download)

## Вариант 2. Windows + виртуальная машина (Ubuntu 24.04)

Если вы хотите познакомиться с Linux и ROS, но не готовы устанавливать отдельную операционную систему, можно использовать виртуальную машину.

Для этого потребуется:

1. Установить одну из программ для виртуализации:

* [`VirtualBox`](https://www.virtualbox.org/wiki/Downloads)
* [`VMware Workstation Pro`](https://disk.yandex.ru/d/jTPgeXTyKQrQiQ)

2. Скачать образ ОС Ubuntu 24.04 LTS: [`Ubuntu 24.04 LTS`](https://releases.ubuntu.com/noble/)

3. Следовать видео-руководству по установке и настройке: `в работе`

4. Установить ROS 2 Jazzy Jalisco (Desktop Install)
   - Следуйте инструкциям на официальном сайте: [`ROS Jazzy Installation Guide`](https://docs.ros.org/en/jazzy/Installation/Ubuntu-Install-Debs.html)

5. Для удобства рекомендуется дополнительно установить:

* Visual Studio Code (с расширением Remote-SSH)
* Terminator — удобный терминал с поддержкой нескольких окон

Этот вариант подходит для обучения и экспериментов без изменения основной системы.

## Вариант 3. Ubuntu 24.04 (рекомендуется)

Наиболее удобный и стабильный вариант для полноценной работы с роботом.

Установка Ubuntu как основной операционной системы обеспечивает максимальную совместимость с ROS 2 и всеми инструментами разработки.

После установки системы необходимо:

Установить **ROS 2 Jazzy (Desktop Install)**, чтобы получить доступ ко всем инструментам разработки, отладки и визуализации.

Для повышения удобства работы рекомендуется установить:

* Visual Studio Code с поддержкой Remote-SSH
* Terminator для работы с несколькими терминалами

Этот вариант рекомендуется для регулярной работы, разработки собственных алгоритмов и участия в проектах.

## Что выбрать

Если вы только начинаете:

**`— используйте Windows для быстрого старта`**

Если хотите разобраться в Linux и ROS:

**`— используйте виртуальную машину`**

Если планируется серьёзная работа с ровером:

**`— используйте Ubuntu как основную систему`**

Такой подход позволит постепенно перейти от базового использования к полноценной разработке.
