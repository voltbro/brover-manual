# Первый запуск

Перед началом работы с **BRover-E5** необходимо выполнить несколько последовательных шагов: проверить устройство, подготовить его к включению и убедиться, что система корректно запускается.

## Подготовка и включение

1. **Распаковка и проверка**

* Достаньте ровер из упаковки
* Проверьте комплектацию (см. раздел [`«Комплектация»`](../10-introduction/10-package-contents.md))
* Убедитесь в отсутствии видимых повреждений

2. **Механическая подготовка**

* Установите колёса, закрепив их винтами `M4×8`

3. **Подключение питания**

* Подключите аккумуляторную сборку
* Перед подключением убедитесь, что она заряжена (используйте кнопку `test` на плате аккумулятора)

Если планируется работа от внешнего блока питания, рекомендуется установить ровер на подставку так, чтобы колёса не касались поверхности.

4. **Подключение к сети**

* Убедитесь, что доступна Wi-Fi сеть, к которой настроен ровер [**по умолчанию**](../10-introduction/40-software.md#сетевые-настройки-по-умолчанию)
* Подключите к этой сети ваш компьютер

5. **Включение устройства**

* Включите ровер кнопкой питания
* Дождитесь завершения загрузки системы

## Проверка работоспособности

После включения рекомендуется убедиться, что основные функции работают корректно.

### Проверка через веб-интерфейс

Самый простой способ проверки — подключиться к веб-интерфейсу и попробовать управлять ровером с клавиатуры.

Инструкция по подключению приведена в разделе [**`Основные возможности`**](../10-introduction/20-key-features.md#веб-интерфейс-управления)


### Проверка управления с джойстика

Ровер также поддерживает управление с радиомодельного пульта.

![Схема джойстика](img/joy.png)

После включения устройство готово к работе с джойстиком.

Для начала движения необходимо правильно установить переключатели и использовать стики управления:

- **CH5 — разрешение движения**

  •	**Нижнее положение** — движение разрешено.

  •	**Среднее и верхнее положения** — движение заблокировано.
- **CH7 — режим скорости**

  •	**Нижнее положение** — минимальная скорость.

  •	**Среднее положение** — средняя скорость.

  •	**Верхнее положение** — максимальная скорость.

- **Левый стик** - движение вперед/назад
- **Правый стик** - повороты влево/вправо

:::warning[Предупреждение]
Если ровер не реагирует на управление, в первую очередь проверьте положение переключателя **CH5** — он должен быть установлен в нижнее положение.
:::
