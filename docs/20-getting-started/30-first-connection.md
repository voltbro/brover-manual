# Первое подключение

После первого включения проверьте, что компьютер и ровер находятся в одной Wi-Fi-сети.

По умолчанию ровер подключается к одной из сетей:

```bash
SSID: TurtleBro / TurtleBro5G
Password: turtlew001
```

## Подключение по SSH

Для первой проверки используйте SSH:

```bash
ssh pi@broverXX.local
```

где `XX` — номер ровера, указанный на корпусе Raspberry Pi.

Если подключение по имени хоста не работает, найдите IP-адрес ровера в интерфейсе роутера и подключитесь по IP:

```bash
ssh pi@<IP-адрес ровера>
```

Учётные данные по умолчанию:

```bash
Логин: pi
Пароль: brobro
```

## Веб-интерфейс

Для управления из браузера откройте:

```bash
http://broverXX.local:8080
```

или:

```bash
http://<IP-адрес ровера>:8080
```

## code-server

Для работы с файлами и кодом через браузер откройте:

```bash
http://broverXX.local:8090
```

или:

```bash
http://<IP-адрес ровера>:8090
```

Пароль по умолчанию:

```bash
brobro
```

Подробные способы подключения описаны в разделе [«Работа с устройством → Подключение к роверу»](../40-device-operation/20-connection-to-rover/10-connection-overview.md).
