# Выключение устройства

Для безопасной работы с **BRover-E5** важно корректно завершать работу системы. Это позволяет избежать повреждения файловой системы и потери данных.

Ниже приведены два сценария: штатное выключение и аварийная остановка питания.

## Выключение через кнопку питания

Если вы работали с файлами на устройстве, рекомендуется сначала завершить операции записи:

```bash
sudo sync
```

Команда `sync` принудительно записывает данные из оперативной памяти на диск.

После этого выполните выключение системы:

```bash
sudo poweroff
```

Дождитесь полного завершения работы операционной системы.

Если вы не выполняли операций с файлами, допустимо использовать упрощённый сценарий:

* нажмите кнопку питания на корпусе ровера
* дождитесь завершения работы

Система корректно завершит работу автоматически.

## Аварийная остановка питания

Для немедленной остановки устройства используется кнопка аварийной остановки.

Этот режим предназначен для экстренных ситуаций, например:

* неконтролируемое движение
* ошибки в управлении
* риск повреждения оборудования

При нажатии аварийной кнопки питание исполнительных систем ровера **немедленно прерывается**.

После использования аварийной остановки рекомендуется:

* проверить состояние системы
* при необходимости перезапустить ровер
* убедиться в корректности работы перед дальнейшим использованием
