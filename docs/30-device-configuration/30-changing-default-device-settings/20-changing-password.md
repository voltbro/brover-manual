# Изменение пароля

На **BRover-E5** по умолчанию используется стандартный пароль пользователя `pi`. В целях безопасности рекомендуется изменить его при первом использовании устройства.

Также при работе через браузер используется пароль для доступа к **code-server**, который при необходимости также можно изменить.

:::note
Стандартный пароль по умолчанию: `brobro`
:::

Для смены пароля вам необходимо знать текущий пароль или иметь права `sudo`.
---

## Изменение пароля пользователя через SSH

Самый простой способ — изменить пароль, подключившись к роверу по SSH.

1. Подключитесь к устройству:

```bash
ssh pi@broverXX.local
```

2. Запустите утилиту `passwd`:

```bash
passwd
```

:::note
При вводе пароля символы не отображаются — это мера безопасности.
:::

3. Следуйте подсказкам:

* Введите текущий пароль.
* Введите новый пароль.
* Подтвердите новый пароль.

---

## Альтернативный метод автоматической смены пароля

Вы можете сменить пароль одной командой (при наличии прав `sudo`):

```bash
echo "pi:новый_пароль" | sudo chpasswd
```

---

## Изменение пароля code-server

Пароль для доступа к **code-server** используется при входе через браузер.

1. Подключитесь к устройству по SSH

2. Откройте файл конфигурации:

```bash
nano ~/.config/code-server/config.yaml
```

3. Найдите параметр:

```yaml
password: your_password
```

4. Измените значение на новый пароль

5. Перезапустите сервис:

```bash
sudo systemctl restart code-server
```

После этого новый пароль будет применён при входе через браузер.

---

## Изменение через скрипт change_pi_params.sh

На устройстве доступен скрипт автоматической настройки параметров:

```bash
./change_pi_params.sh
```

Скрипт позволяет:

* изменить имя хоста
* изменить пароль пользователя `pi`
* обновить связанные параметры системы

В процессе выполнения скрипт последовательно запросит:

* новое имя устройства
* новый пароль

После завершения рекомендуется перезагрузить устройство:

```bash
sudo reboot
```

## Проверка

Попробуйте подключиться к устройству с новым паролем:

```bash
ssh pi@broverXX.local
```

Если вход выполняется успешно — пароль изменён корректно.
